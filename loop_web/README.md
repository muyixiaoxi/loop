# Loop Web Application

这是一个基于 React 19 + TypeScript + Vite 构建的现代化 Web 应用，集成了以下主要技术栈：

## 技术栈

- **前端框架**: React 19
- **状态管理**: MobX 6
- **UI 组件库**: Ant Design 5
- **路由**: React Router DOM 7
- **HTTP 客户端**: Axios
- **构建工具**: Vite
- **CSS 预处理器**: Sass
- **数据库**: Dexie (IndexedDB)
- **工具库**:
  - Crypto-js (加密)
  - UUID (唯一标识符)
  - Pinyin (拼音转换)

## 开发脚本

```bash
# 启动开发服务器
npm run dev

# 构建生产版本
npm run build

# 预览生产版本
npm run preview

# 运行 ESLint 检查
npm run lint
```
